# Android第三次实验
=======================================================
自定义WebView验证隐式Intent的使用
 本实验通过自定义WebView加载URL来验证隐式Intent的使用。

1. 第一个应用：获取URL地址并启动隐式Intent的调用。
2. 第二个应用：自定义WebView来加载URL
